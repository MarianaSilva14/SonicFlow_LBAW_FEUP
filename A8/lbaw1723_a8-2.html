A8: Vertical prototype

1. Implemented Features

1.1. Implemented User Stories

The user stories that were implemented in the prototype are described in the following table.

User Story reference Name Priority Description
US03 About page high As a Visitor, I want to access about page, so that I can see a complete website's description.
US04 Contact page high As a Visitor, I want to access contact page, so that I can see every contact.
US07 FAQ page high As a Visitor, I want to access FAQ page, so that I can see FAQ's.
US09 Login high As a Visitor, I want to authenticate into the system, so that I can access privileged information.
US010 Register high As a Visitor, I want to register myself into the system, so that I can authenticate myself into the system.
US52 Edit Profile high As an Owner, I want to change my information, so that I can keep it updated (e.g. password,address,profile picture).
US54 Favorites add high As an Owner, I want to add an item to my favorites.
US55 Favorites remove high As an Owner, I want to remove an item from my favorites list.
